# Aditya Kumar — Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, Three.js, and Framer Motion.

## Tech Stack

- **React 18** + **TypeScript** — UI framework
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **Three.js / R3F** — 3D scenes
- **Supabase** — AI Chat backend

## Getting Started

```sh
git clone https://github.com/adityadhimann/updatedportfolio.git
cd updatedportfolio
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) to view.
